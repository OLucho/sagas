# Sagas Frontend

Next.js app for the Sagas Pokemon TCG collection manager. Browse sets, track your collection, create lists, and share them with customers.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- shadcn/ui component library (Radix UI + Tailwind)
- SWR for data fetching
- Lucide icons
- Vercel Analytics
